# spayon-onboarding

Internal merchant onboarding system for Spayon (payment processor settling in USDT).

Project knowledge: `.claude/skills/project-knowledge/references/`
Default branch: `main`

## Quick reference

- Dev: `npm run dev`
- Seed admin: `npm run seed` (requires `ADMIN_PASSWORD` in `.env`)
- Deploy: push to `main` → Railway auto-deploys via Dockerfile
- Admin login: nastya@spayon.io
